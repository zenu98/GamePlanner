# Project

게임을 좋아하는 초보개발자 둘이서 진행했습니다.<br> 국내에 게임 일정을 알려주는 **모바일 앱**이 없어서 아쉬웠기에 저희가 직접 만들어 보았습니다.<br>
기다리던 게임들을 한눈에 **플랫폼**, **장르**별로 확인할 수 있는 앱입니다.<br>
올해의 **기대작**들을 미리미리 알아보고 자세한 정보를 확인하세요!
